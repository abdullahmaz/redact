Lethe

Interactive proof-of-concept for UNSIR — Fast Yet Effective Machine Unlearning. Pick the classes you want a trained ResNet-18 to forget. Watch it forget them, in seconds, without ever revisiting a single forget-class sample.

device: ? checkpoint: ?

1. Choose classes to forget

Tick one or more CIFAR-10 classes. Their training samples are never shown to the optimiser during unlearning.

2. Per-class accuracy

Baseline = original model. Current = after the latest unlearn run. Hatched = forget classes from the last run.

3. Phase timing

Wall-clock time of each UNSIR phase. The paper claims under ~4 s end-to-end on ResNet-18 / CIFAR-10.

4. Sample predictions

A handful of test images run through the current model. After unlearning, forget-class images should be confidently mis-classified into a retain class.
